# Contributing to Tattoo Pain Map & Guide

Thank you for your interest in contributing to the Tattoo Pain Map & Guide! This tool is part of the Poli International Widget Suite, built for the body art community.

## 🛠️ Development Setup

1. **Fork and clone the repository**

   ```bash
   git clone https://github.com/Poli-International/pain-guide.git
   ```

2. **No Build Process**
   This tool is built with pure HTML/CSS/JS. Open `index.html` in your browser to test.
